### Separatrix Locator

A tool for locating separatrices in black-box dynamical systems.

---

This is the code accompanying the paper [Finding separatrices of dynamical flows with Deep Koopman Eigenfunctions](https://www.arxiv.org/abs/2505.15231).

Please find **[a newer version of the code here](https://github.com/KabirDabholkar/separatrix_locator)**.
